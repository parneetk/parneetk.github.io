Relevant Academic Projects:
Evaluation of Branch Prediction Strategies, worked in a group of 3
Evaluated three existing branch prediction strategies for instruction level parallelism using Simplescalar simulator on five benchmarks. Designed and developed two new strategies, which improved performance on three benchmarks.
Panorama Creation
360° panoramic image creation involving 1) projecting images to cylindrical coordinates, 2) pairwise feature matching using SIFT and RANSAC, 3) pairwise translation motion estimate, 4) warping and pyramid blending.
Study of pattern recognition techniques in classifying gene expression profiles
Implemented principal component analysis, multidimensional scaling and isometric mapping to reduce dimensionality of dataset and analyzed the impact of dimensionality reduction on support vector machines, decision trees, k-nearest neighbors, neural networks, bagging and ensemble learners.
2D Motion Estimation
Implemented Horn & Schunck and Lucas Kanade algorithms for 2D motion estimation including hierarchical implementation with pyramids.
Fun with images using Homography
Point correspondences found automatically using Harris corner detector or SIFT features were used to compute Homography robustly using RANSAC. Homography obtained was used to 1) implement image stitching, 2) project an image in a user specified plane on a target image.
Concrete texture analysis for micro-crack detection, worked in a group of 2
Assembled and aligned a texture camera to capture images and measure bidirectional reflectance distribution function. An intensity histogram based approach was proposed to classify crack and non-crack regions.
Real Time Hand Gesture Recognition and Blink Detection, worked in a group of 2
Implemented hand gesture recognition using color segmentation and contours for controlling robot/mouse movement. Blink detection was implemented using blob detection. Implemented in C using OpenCV.
Decision System for Air Ticket Purchase, worked in a group of 3
Developed an airline-price trend analysis system using Autoregressive Integrated Moving Average and Double Exponential Moving Average.
Face Recognition using Principal Component Analysis
Implemented Principal Component Analysis for face recognition to demonstrate the application of optimum signal processing in the field of computer vision.
